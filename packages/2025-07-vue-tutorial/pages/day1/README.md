# Vue 3 教學 - Day 1

## 課程目標

- 建立開發環境
- 了解 Vue 3 基本概念
- 學習資料驅動畫面的原理

## 課程內容

### 一、開發環境建立

- 安裝 Node.js 和 npm/pnpm
- 使用 Vite 建立 Vue 3 專案
- 開發工具設定（VS Code 擴充功能）

### 二、Vue 3 簡介

- Vue 3 的新特性
- Composition API vs Options API
- 響應式系統原理

### 三、資料驅動畫面

- 模板語法
- 資料綁定
- 事件處理

## 範例組件

### HelloVue.vue

基本的 Vue 3 組件範例，展示：

- `ref()` 響應式資料
- 模板插值 `{{ }}`
- 事件綁定 `@click`

## 練習作業

1. 建立一個計數器組件
2. 嘗試不同的資料綁定方式
3. 添加樣式和動畫效果

## 執行投影片

```bash
npm run dev:day1
```
